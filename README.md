**CCNA CLI Practice Tool**  

A lightweight, offline, command-line application designed to help students prepare for the Cisco CCNA
(200-301) exam through structured practice quizzes, performance tracking, and targeted review of weak
areas.

Key features:
1. Generate 20 random multiple-choice questions from a selected CCNA domain.
2. Take a 50-question mixed exam covering all CCNA domain.
3. View scores, percentages, and pass/fail status immediately after each quiz.
4. Review only the questions you answered incorrectly, along with the correct answers.
5. Track historical performance per syllabus area and identify strengths and weaknesses.

Supported CCNA Domains (200-301):
* Network Fundamentals
* Network Access
* IP Connectivity
* IP Services
* Security Fundamentals
* Automation and Programmability
